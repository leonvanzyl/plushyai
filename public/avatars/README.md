# Avatar Images

This directory contains avatar images for user profiles and testimonials.

## Required Images

### User Avatar
- `user-1.jpg` - Main user avatar for mock user
- Recommended size: 200x200px
- Format: JPG or PNG

### Testimonial Avatars
- `testimonial-1.jpg` through `testimonial-4.jpg`
- Avatars for testimonial users on landing page
- Recommended size: 80x80px or 100x100px
- Format: JPG or PNG

## Placeholder Images

For development, you can:
1. Use avatar placeholder services like https://ui-avatars.com/
2. Use stock photo services
3. Generate simple colored circles with initials

## Usage

These images are referenced in:
- `src/lib/mock-data.ts` for the `mockUser` and `mockTestimonials`
- Various components that display user profiles
