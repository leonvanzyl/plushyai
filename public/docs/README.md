# Documentation Images

This directory contains screenshots and images for the documentation pages.

## Recommended Images

### Getting Started Screenshots
- `upload-screenshot.png` - Screenshot of upload interface
- `generation-progress.png` - Screenshot of generation in progress
- `result-view.png` - Screenshot of final result

### How-to-Use Screenshots
- `dashboard-screenshot.png` - Dashboard overview

### Tips & Examples
- `good-photo-example.jpg` - Example of a good photo for plushie generation
- `bad-photo-example.jpg` - Example of what to avoid

## Format

- Screenshots: PNG format preferred (supports transparency)
- Example photos: JPG or PNG
- Recommended width: 800-1200px for screenshots
- Compress images to reduce file size

## Usage

These images will be used in:
- Documentation pages under `/docs/*`
- Tutorial sections
- FAQ answers with visual aids
